# 🎮 VALORANT Stretched Resolution Configurator (4:3)

Una herramienta de escritorio moderna, rápida y segura diseñada para jugadores de VALORANT que buscan optimizar su ventaja competitiva utilizando resoluciones estiradas (4:3) sin complicaciones técnicas.

## ✨ Características Principales

*   **👥 Selector Multicuenta (Nuevo):** Detecta todas las cuentas registradas en tu PC y te permite alternar entre ellas mediante un menú desplegable interactivo.
*   **🔗 Acceso Directo Dinámico (Nuevo):** Haz clic en el texto de la cuenta detectada para abrir directamente el archivo de configuración `.ini` específico de ese perfil.
*   **🖥️ Sincronización con Windows (Nuevo):** Opción para cambiar la resolución de tu monitor a la par de la del juego con un solo botón, eliminando el molesto parpadeo negro al hacer Alt+Tab.
*   **🚀 Impulso FPS Ultra (Mejorado):** Fuerza parámetros gráficos ultra bajos no disponibles en los menús convencionales. Ahora incluye un menú emergente para aplicar valores predeterminados o personalizar cada parámetro del 0 al 5 con memoria inteligente de guardado (JSON).
*   **🛡️ Bloqueo Anti-Reversión:** Opción para poner el archivo en "Solo Lectura" y evitar que el juego borre tu configuración (con advertencia de seguridad para parches del juego).
*   **💾 Sistema de Backup:** Crea y restaura copias de seguridad de tus ajustes anteriores con un solo clic.
*   **🎨 Calidad 3D Dinámica:** Slider interactivo para ajustar la nitidez del juego con indicadores visuales de color.
*   **Lógica de Inyección Segura:** Modifica el archivo `GameUserSettings.ini` de forma precisa, respetando mayúsculas y evitando la duplicidad de líneas.
*   **Multi-idioma Dinámico:** Selector en tiempo real entre Español e Inglés.

## 🚀 Cómo usar

1.  **Cierra VALORANT** por completo.
2.  Descarga el ejecutable desde la sección de [Releases](https://github.com/MauricioEsparron/configurador_valorant/releases).
3.  Ejecuta el programa como **Administrador**.
4.  Selecciona tu cuenta, ajusta la resolución y decide si deseas sincronizarla con Windows.
5.  Haz clic en **APLICAR CONFIGURACIÓN**.
6.  ¡A jugar! 🎯

## 🛠️ Tecnologías utilizadas

*   **Python 3.13**
*   **CustomTkinter:** Para la interfaz gráfica moderna y oscura.
*   **Pillow (PIL):** Para el procesamiento dinámico y renderizado de iconos.
*   **Windows API (ctypes):** Para la manipulación segura de la memoria y resoluciones de pantalla a nivel de sistema.
*   **PyInstaller:** Para la compilación limpia a ejecutable nativo sin consola activa.

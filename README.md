# 🎮 VALORANT Stretched Resolution Configurator (4:3)

Una herramienta de escritorio moderna, rápida y segura diseñada para jugadores de VALORANT que buscan optimizar su ventaja competitiva utilizando resoluciones estiradas (4:3) sin complicaciones técnicas.

## ✨ Características Principales

*   **Detección Inteligente de Cuenta:** Localiza automáticamente la carpeta de configuración de tu última cuenta activa (soporte para múltiples regiones: LATAM, BR, NA, etc.).
*   **Lógica de Inyección Segura:** Modifica el archivo `GameUserSettings.ini` de forma precisa, evitando duplicidad de líneas y errores de formato.
*   **🚀 Impulso FPS Ultra (Nuevo):** Fuerza parámetros gráficos ultra bajos que no están disponibles en los menús normales para maximizar el rendimiento.
*   **🛡️ Bloqueo Anti-Reversión (Nuevo):** Opción para poner el archivo en "Solo Lectura" y evitar que el juego borre tu configuración.
*   **💾 Sistema de Backup (Nuevo):** Crea y restaura copias de seguridad de tus ajustes anteriores con un solo clic.
*   **🎨 Calidad 3D Dinámica (Nuevo):** Slider interactivo para ajustar la nitidez del juego con indicadores visuales de color.
*   **Multi-idioma Dinámico:** Selector en tiempo real entre Español e Inglés.

## 🚀 Cómo usar

1. **Cierra VALORANT** por completo.
2. Descarga el ejecutable desde la sección de [Releases](https://github.com).
3. Ejecuta el programa como **Administrador**.
4. Selecciona tu resolución o ingresa una personalizada.
5. Haz clic en **APLICAR CONFIGURACIÓN**.
6. ¡A jugar! 🎯

## 🛠️ Tecnologías utilizadas

*   **Python 3.13**
*   **CustomTkinter:** Para la interfaz gráfica moderna.
*   **Regex (Expresiones Regulares):** Para la manipulación precisa de archivos de configuración.
*   **PyInstaller:** Para la compilación a ejecutable (.exe).
